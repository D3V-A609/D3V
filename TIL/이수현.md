## 2025.02.04 TIL

### Today's Keywords
`Redux 설치 및 초기 설정`, `Redux Slice 통합 및 리팩토링`, `Redux와 API 연동`, `Redux Router 상태 관리`, `Card 컴포넌트 스타일링`

### 회고

오늘은 Redux를 활용한 상태 관리 리팩토링을 진행했습니다. 주요 작업은 다음과 같습니다:

1. **상태 관리 통합**: 기존의 분리되어 있던 일일 질문과 전체 질문 상태를 하나의 통합된 questionSlice로 결합했습니다. 이를 통해 코드의 중복을 줄이고 관리를 용이하게 만들었습니다.

2. **API 연동**: Redux Toolkit을 사용하여 오늘의 질문 API를 연동했습니다. 이를 통해 비동기 데이터 처리를 더욱 체계적으로 관리할 수 있게 되었습니다.

3. **라우팅 개선**: Redux와 React Router를 통합하여 라우팅 상태를 Redux store에서 관리하도록 개선했습니다. 이로써 애플리케이션의 네비게이션 상태 관리가 더욱 예측 가능해졌습니다.

4. **UI 개선**: Card 컴포넌트의 카테고리 배경 색상을 수정하여 시각적 일관성을 향상시켰습니다.

이번 리팩토링을 통해 상태 관리 구조가 더욱 체계적으로 개선되었으며, 향후 기능 확장에도 더욱 유연하게 대응할 수 있게 되었습니다.
